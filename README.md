# JDBC_Servlet_Maven_Work
This repository contains a collection of core Java web applications developed using JDBC, Servlets, Maven, and Hibernate, focusing on CRUD operations and MVC-based architecture.
These projects are built without Spring or Spring Boot to strengthen understanding of low-level Java web development concepts.
